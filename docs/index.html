yt-downloader

Roda inteiramente no GitHub — seu celular só envia a ordem e recebe o link do arquivo pronto.

Limites desse esquema (leia antes de baixar algo grande)
Tempo por jobaté 6h por execução — 4K muito longo pode estourar
Minutos/mês grátis2.000 min — ≈400 vídeos de 5min ou ≈20 vídeos de 1h
Tamanho por arquivo2 GB no Artifact · Releases aceitam arquivos maiores, mas ainda têm limite prático
ArmazenamentoRelease não expira; o Artifact (backup) some em 90 dias
Bloqueio de IPYouTube às vezes bloqueia os servidores do GitHub — os cookies ajudam, não garantem
Validade dos cookiesSessão do Google expira/rotaciona; de tempos em tempos será preciso exportar e trocar o arquivo

Sobre o arquivo de cookies: ele fica no repositório, não como Secret. Isso é mais simples de trocar, mas significa que qualquer pessoa com acesso de leitura ao repositório consegue ler suas credenciais de sessão do Google/YouTube. Mantenha o repositório privado e nunca publique a pasta cookies/ pelo GitHub Pages.
